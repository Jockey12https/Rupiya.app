## Tone & Voice

### Characteristics
- **Trustworthy**: Professional, reliable language
- **Empowering**: Positive, encouraging tone
- **Educational**: Clear, informative content
- **Bilingual**: Hindi + English for accessibility
- **Farmer-Centric**: Focused on farmer benefits
- **Action-Oriented**: Clear CTAs and next steps

### Language Guidelines
- Use simple, clear language
- Avoid jargon where possible
- Explain technical terms
- Use active voice
- Focus on benefits, not features
- Include social proof (testimonials, numbers)
